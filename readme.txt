SKU Search provides users the ability to search products by SKU.  This plugin is compatible with WooCommerce.

== Description ==

SKU Search allow users to search products by SKU.  This plugin is compatible with product variations too, and will automatically fill in the attribute menus on searched product page.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The plugin is now working!

== Frequently Asked Questions ==

= Does this plugin support searching SKUs of variations? =

Absolutely!  This plugin will also fill in the attribute dropdowns so your customers won't have to.  Of course, this plugin supports simple products as well.

= Does this plugin use cookies? =

SKU Search stores a single cookie, 'sku', for a short time in order to lookup the variation searched once a customer arrives at the given product page after a search.
